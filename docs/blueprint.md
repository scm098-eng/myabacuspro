# **App Name**: AbacusAce

## Core Features:

- Homepage Navigation: Provide a home page with links to practice tests, 'about us', and 'contact us'.
- Automatic Question Progression: Display math questions automatically, one after another after the previous is answered or skipped.
- Multiple Choice Interface: Show multiple-choice options for each question, and evaluate each answer for correctness.
- Addition/Subtraction Tests: Include addition and subtraction tests with a mix of question types and difficulty levels.
- Multiplication/Division Tests: Incorporate multiplication and division tests with single, double, triple, and four-digit numbers.
- Timed Testing: Implement a timer for each test (easy, medium, hard), with automatic submission after the timer ends and the option for manual submission.
- User Profile system: Implement a login system and create student profiles (without database storage of this information).

## Style Guidelines:

- Primary color: Calming blue (#64B5F6), reminiscent of focus and learning environments.
- Background color: Light grayish-blue (#ECEFF1), a gentle and non-distracting backdrop.
- Accent color: Warm orange (#FFB74D), to highlight correct answers and important UI elements.
- Body and headline font: 'Inter' (sans-serif), providing a modern and readable typeface for questions and instructions.
- Use clean, geometric icons for navigation and representing different mathematical operations.
- Design a distraction-free interface that minimizes visual clutter, ensuring users stay focused on the questions and timer.
- Incorporate subtle transitions and animations when switching between questions, providing a seamless user experience.
